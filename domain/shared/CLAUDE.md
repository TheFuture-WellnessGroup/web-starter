# Shared Domain

도메인 간 공유되는 코드를 관리합니다.

## 용도

- 여러 도메인에서 사용하는 공통 타입
- 도메인 간 통신을 위한 인터페이스
- 공통 비즈니스 규칙 (예: 금액 계산, 날짜 처리)

## 규칙

1. **특정 도메인에 종속되지 않는 코드만**
   - 특정 도메인 전용 코드는 해당 도메인 폴더에 배치

2. **최소한으로 유지**
   - 무분별한 공유 코드 추가 지양
   - 정말 공유가 필요한 경우에만 추가

3. **폴더 구조**
   ```
   shared/
   ├── lib/        # 공통 유틸리티
   ├── types/      # 공통 타입
   └── index.ts
   ```
